# Security Policy

Per segnalazioni di sicurezza o privacy usare un canale riservato e non
issue pubbliche.

## Regole minime

- Non caricare file reali di clienti in issue pubbliche.
- Non allegare atti, fascicoli, buste PCT reali o report contenenti dati
  sensibili.
- Non inserire token, credenziali o altri segreti nel repository.
- Segnalare vulnerabilita', problemi privacy o esposizioni accidentali
  tramite canale riservato.
- Verificare sempre output e report prima dell'uso professionale.

## Ambito

Il progetto aiuta nella validazione preliminare dei file, ma non
sostituisce il controllo professionale finale dell'utente.
